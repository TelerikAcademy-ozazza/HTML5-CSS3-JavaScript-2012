    2. What is HTML?

    HTML is a languge for describing web pages.

    • HTML stands for Hyper Text Markup Language

    • HTML is not a programming lanuage, it is a markup language

    • A markup lanuage is a set of markup tags

    • HTML uses markup tags to describe web pages

  2. Get Started

    3. Editing HTML

    In this tutorial we use a plain text editor (like Notepad) to edit HTML. We believe this is the best way to learn HTML.

    However, professional web developers often prefer HTML editors like Dreamweaver, instead of writing plain text.
